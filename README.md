# Museum of Fine Art

[![Codacy Badge](https://api.codacy.com/project/badge/Grade/1fdaaafff33d473b8777a674b4911781)](https://app.codacy.com/gh/KateCodesZ/native-js-museum?utm_source=github.com&utm_medium=referral&utm_content=KateCodesZ/native-js-museum&utm_campaign=Badge_Grade)

A learning project featuring a virtual museum showcasing exhibitions from three different artists, built with vanilla JavaScript.

## Overview

This project is an interactive museum web application that displays artworks from the Metropolitan Museum of Art collection. It includes artist exhibitions, data visualizations, and an art knowledge quiz to engage visitors.

## Features

- **Artist Exhibitions** - Explore curated collections from three different artists
- **Metropolitan Museum of Art API Integration** - Real-time artwork data and images
- **Interactive Data Visualizations** - Charts and statistics powered by Chart.js
- **Art Quiz** - Test your knowledge with a 10-question quiz about art and artists

## Technologies Used

- **Vanilla JavaScript** - No frameworks, pure JS for DOM manipulation and logic
- **Metropolitan Museum of Art API** - Artwork data source
- **Chart.js** - Data visualization library
- **HTML5 & CSS3** - Structure and styling

## Learning Objectives

This project was created to practice:
- Working with external APIs and asynchronous JavaScript
- DOM manipulation without frameworks
- Data visualization techniques
- Building interactive user experiences
- Organizing vanilla JavaScript code
